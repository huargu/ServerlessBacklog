# Serverless Sprint Backlog Manager

# Features

- Create sprint
- Create backlog Item
- Move item into any sprint
- Delete backlog item

## Backend

To deploy and run the application:

```
cd backend
npm install
sls deploy -v
```

## Frontend

To build and run the application after edition config file in 'client/src/config.ts':
```
cd client
npm install
npm run start
```
